# 🌿 Calmora

> **"Find your calm. Find your power."**

Calmora is an AI-powered mental wellness and personal growth platform designed to help users cultivate mindfulness, build positive habits, maintain daily discipline, and improve overall emotional well-being.

---

## ✨ Key Features

- 🤖 **AI Companion (`/ai-companion`)**: A supportive, non-judgmental AI chatbot for emotional support and guided reflection.
- 📓 **Interactive Journal (`/journal`)**: Expressive journaling with mood tracking, tags, and daily prompts.
- 📊 **Mood & Habit Tracking (`/habits`)**: Log daily emotional states and track habit streaks with interactive analytics.
- 🌸 **21-Day Transformational Challenge (`/challenges`)**: Structured daily mindfulness challenges to build positive habits.
- 💨 **Relax & Meditation Space (`/relax`)**: Guided breathing exercises, ambient soundscapes, and meditation timers.
- 📈 **Wellness Analytics & Reports (`/reports`)**: Visual progress indicators, calm scores, and streak tracking.
- 👥 **Community & Student Hubs (`/community`, `/student`)**: Peer support, shared milestones, and tailored resources for students.
- 🚨 **Emergency Support (`/emergency`)**: Immediate access to crisis hotlines and grounding exercises.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) + Custom Glassmorphism UI
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### 1. Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (v18 or higher) installed.

### 2. Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/0251cys075/calmora.git
cd calmora
npm install
```

### 3. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or [http://localhost:3001](http://localhost:3001)) in your browser to view the application.

### 4. Build for Production

```bash
npm run build
npm run start
```
